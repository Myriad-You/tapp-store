# 便携便签

简单高效的笔记工具，快速记录想法和灵感。

## 功能特点

- 📝 **快速笔记** - 一键添加新笔记，支持快捷键
- 🔍 **即时搜索** - 快速筛选和找到需要的笔记
- 🗑️ **轻松管理** - 删除或清空笔记
- ⏰ **时间戳** - 可选显示创建时间
- 📱 **多尺寸小组件** - 支持 2x2、4x2、4x4 三种尺寸
- 🌙 **深色模式** - 完美适配系统主题
- 🌐 **多语言支持** - 中文、英文、日文

## 架构

```
com.myriad.quick-notes/
├── manifest.json        # 应用配置
├── main.js              # 核心逻辑（Widget + Page）
├── widget.css           # 小组件样式
├── page.css             # 页面样式
├── widget-2x2.html      # 2x2 小组件模板
├── widget-4x2.html      # 4x2 小组件模板
├── widget-4x4.html      # 4x4 小组件模板
├── page.html            # 页面模板
└── README.md            # 说明文档
```

## 技术特性

- **分离式 CSS 架构** - Widget 和 Page 使用独立样式文件
- **HTML 模板系统** - 不同尺寸使用专用 HTML 模板
- **国际化支持** - 内置 i18n 系统
- **响应式设计** - 使用 CSS Container Queries
- **Glass Morphism** - 毛玻璃视觉效果
- **动态光效** - 基于壁纸主色的光晕装饰

## 权限说明

| 权限              | 用途              |
| ----------------- | ----------------- |
| `storage`         | 保存笔记内容      |
| `ui:notification` | 操作成功提示      |
| `ui:theme`        | 适配深色/浅色主题 |

## 设置选项

| 设置项       | 类型   | 默认值 | 说明               |
| ------------ | ------ | ------ | ------------------ |
| `maxNotes`   | number | 100    | 最大笔记数量       |
| `showTimestamp` | toggle | true  | 显示创建时间    |
| `saveHistory` | toggle | true  | 保存历史记录      |
