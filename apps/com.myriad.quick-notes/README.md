# 便携便签

简单的笔记工具，快速记录想法和灵感。

## 功能特点

- 📝 **快速笔记** - 一键添加新笔记
- 🔍 **搜索功能** - 快速找到需要的笔记
- 🗑️ **轻松管理** - 删除不需要的笔记
- ⏰ **时间戳** - 可选显示创建时间
- 📱 **小组件支持** - 桌面小组件快速访问

## 权限说明

| 权限              | 用途              |
| ----------------- | ----------------- |
| `storage`         | 保存笔记内容      |
| `ui:notification` | 操作成功提示      |
| `ui:theme`        | 适配深色/浅色主题 |
| `widget:register` | 注册桌面小组件    |
| `component:page`  | 注册独立页面      |
